## 🎮 使用方法

### 基础使用
1. **点击英文文本**: 直接点击蓝色的英文文本即可播放语音
2. **语音控制**: 使用右下角的控制按钮
   - 🔊 开关语音功能
   - ⏹️ 停止当前播放
   - ⚙️ 打开语音设置

### 语音设置
1. **选择语音**: 从下拉菜单选择您喜欢的语音（推荐⭐标记的高质量语音）
2. **调整参数**: 
   - **语速**: 0.5x - 1.5x（推荐0.85x）
   - **音调**: 0.8 - 1.3（推荐1.0）
   - **音量**: 30% - 100%（推荐80%）
3. **测试效果**: 点击"测试语音效果"按钮

### 键盘快捷键
- `Ctrl/Cmd + S`: 切换语音功能
- `Ctrl/Cmd + X`: 停止语音播放
